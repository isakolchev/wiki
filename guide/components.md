# Компоненты

Страница для проверки всех цветных блоков и элементов оформления.

## Цветные блоки

::: info Инфо
Нейтральный информационный блок.
:::

::: tip Подсказка
Блок с акцентным (бренд) оттенком.
:::

::: warning Предупреждение
Тёплый блок-предупреждение — обрати внимание, но это не ошибка.
:::

::: danger Ошибка
Блок ошибки — для критичных сообщений.
:::

## Таблица

| Элемент | Назначение |
|---------|------------|
| Заголовки | структура страницы |
| Ссылки | переходы и акценты |
| Код | моноширинный шрифт |
| Блоки | статусы и уведомления |

## Код

```ts
type Status = 'info' | 'tip' | 'warning' | 'danger'

function badge(status: Status): string {
  return `badge--${status}`
}
```
