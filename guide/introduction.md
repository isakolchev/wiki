# Введение

Это демонстрационная документация для предпросмотра темы. Весь текст здесь нейтральный — он нужен только для того, чтобы показать, как выглядят типографика, ссылки, таблицы и блоки в текущем оформлении.

> Цитата-блок: тема настраивается через CSS-переменные, шрифты и цвета задаются в одном месте.

Можно использовать **жирный текст**, *курсив*, `инлайн-код` и [ссылки](#). Ниже на других страницах — таблицы, блоки-уведомления и примеры кода.

## Что внутри

- Введение — эта страница
- Быстрый старт — первые шаги
- Конфигурация — параметры и пример настройки
- Компоненты — цветные блоки и элементы
- Вопросы и ответы — частые вопросы

## Зачем это нужно

Набор страниц подобран так, чтобы задействовать максимум элементов оформления: заголовки разных уровней, списки, таблицы, код и уведомления четырёх типов.
