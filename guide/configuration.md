# Конфигурация

Демонстрация таблицы параметров, блока кода и блока-предупреждения.

## Параметры

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `title` | string | — | Заголовок проекта |
| `lang` | string | `ru-RU` | Язык интерфейса |
| `theme` | string | `auto` | Светлая, тёмная или авто |
| `compact` | boolean | `false` | Плотная раскладка |

## Пример

```js
export default {
  title: 'Демо',
  lang: 'ru-RU',
  theme: 'auto',
  compact: false
}
```

::: warning Предупреждение
Меняйте по одному параметру за раз и проверяйте результат — так проще понять, что на что влияет.
:::
