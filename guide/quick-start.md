# Быстрый старт

Короткая последовательность шагов — чтобы показать нумерованные списки и блок-подсказку.

1. Откройте проект и установите зависимости.
2. Запустите локальный сервер разработки.
3. Откройте страницу в браузере и вносите правки — они применяются на лету.

```bash
npm install
npm run dev
```

::: tip Подсказка
Большинство визуальных параметров меняется в одном файле стилей — цвета и шрифты подхватываются без перезапуска.
:::

## Дальше

После первого запуска переходите к разделу «Конфигурация», чтобы настроить параметры под себя.
