# Quotcat Wiki

База знаний и документация проекта на [VitePress](https://vitepress.dev). Репозиторий служит предпросмотром темы оформления (цвета, шрифты, светлая и тёмная) и демонстрацией элементов документации.

## Запуск

```bash
npm install
npm run docs:dev      # локальный сервер: http://localhost:5173
npm run docs:build    # сборка статики в .vitepress/dist
npm run docs:preview  # предпросмотр собранной статики
```

## Структура

```
.vitepress/
  config.js            # навигация и настройки сайта
  theme/
    index.js           # точка входа темы
    theme.css          # дизайн-токены (цвета, шрифты, отступы)
    custom.css         # мост: переменные VitePress ← токены + правки оформления
guide/                 # статьи документации
public/                # статические файлы (логотипы)
index.md               # главная
```

## Тема

Цвета и шрифты задаются через CSS-переменные. Основная настройка — в `.vitepress/theme/custom.css`, который переопределяет переменные VitePress значениями из `theme.css`. Светлая и тёмная темы настраиваются отдельно.
